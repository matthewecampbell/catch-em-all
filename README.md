# Catch 'Em All!

* `git clone https://github.com/rwarbelow/catch-em-all.git`
* `rake db:create db:migrate db:seed`
* `rails s`
* navigate to `http://localhost:3000`
* Want hard mode? Uncomment the JS file to make your Pokemon run around.
